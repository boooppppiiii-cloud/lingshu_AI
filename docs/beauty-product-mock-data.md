## Agent 调用建议

- 商品发布 Agent：优先调用品名、规格、MOQ、交期、认证资料、禁用功效词和目标市场。
- 流量专家：默认使用主要语言生成英语内容；进入中东素材时优先尝试阿拉伯语版本。
- 转化专家：报价前必须确认目标市场、包装需求、MOQ、认证文件和上架渠道。
- 广告/内容 Agent：优先使用“成分 + 肤感 + 私标 + 快速打样”的内容结构。
- 经营分析 Agent：重点观察私标询盘占比、样品单转大货率、不同语言脚本回复率。
