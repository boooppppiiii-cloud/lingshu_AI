============================================================
  海外营销工作台 - 打包说明
============================================================

【需要打包的文件/文件夹】（必须全部带上）
------------------------------------------------------------
✅ src/                  前端源码
✅ server/               后端源码
✅ public/               静态资源
✅ scripts/              工具脚本
✅ pb_data/              数据库文件（账号、数据）
✅ pb_migrations/        数据库迁移
✅ .env                  环境变量（API Key 等）
✅ pocketbase.exe        数据库程序
✅ package.json
✅ package-lock.json
✅ tsconfig.json
✅ vite.config.ts
✅ index.html
✅ ecosystem.config.cjs  PM2 配置
✅ 启动工具.bat          一键启动脚本

【不需要打包】（可以省略，节省空间）
------------------------------------------------------------
❌ node_modules/         依赖库（目标机器会自动安装）
❌ dist/                 构建产物（不需要）
❌ .git/                 版本历史（不需要）

【目标电脑需要预先安装】
------------------------------------------------------------
1. Node.js 20+（下载：https://nodejs.org/）
   安装后重启电脑

【首次运行步骤】
------------------------------------------------------------
1. 把文件夹复制到目标电脑任意位置
2. 双击运行「启动工具.bat」
3. 首次运行会自动安装依赖（需要约5分钟）
4. 安装完成后浏览器会自动打开 http://localhost:5174

【之后每次使用】
------------------------------------------------------------
直接双击「启动工具.bat」即可

【注意事项】
------------------------------------------------------------
- 每次使用都要保持「启动工具.bat」窗口在后台运行
- 如果端口被占用，检查是否有其他程序占用 5174/8790/8091 端口
- .env 里的 API Key 不用改，已经配置好了
